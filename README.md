# **Discourse Topic Gallery**

## Description

This topic adds a `/gallery` route to each topic.
The gallery displays all the pictures from the topic.

⚠️ It was made using AI and the code was barely reviewed. Don't install it if you can't review the code and decide it's safe to use. ⚠️

## Instructions

Enable the plugin after installation.
By default, galleries are available only for admins. You can change this group-specific setting.
